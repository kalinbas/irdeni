# Irdeni Web

The web port lives here. For the full project overview, screenshots, and live game link, see the repository README at [`../README.md`](../README.md).

## Commands

```bash
npm ci
npm run dev
npm run build
npm run lint
```

## Notes

- `npm run dev` and `npm run build` both extract fresh content from the original QuickBasic files before starting
- The production build is deployed to GitHub Pages through `.github/workflows/deploy.yml`
